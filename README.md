# 🚚 利克健身集团官网管理系统

## ✨ 功能特性

### 🏢 内容管理
- **公告系统** - 发布公司新闻和重要通知
- **品牌介绍** - 展示企业文化和品牌形象
- **门店清单** - 展示各地门店信息和联系方式
- **教练介绍** - 教练团队风采展示
- **多媒体中心** - 图片和视频展示区
- **邮品清单** - 特殊项目展示区

### ⚙️ 系统设置
- **LOGO管理** - 自定义公司LOGO
- **背景设置** - 个性化网页背景

### 🔒 安全特性
- 所有管理操作需密码验证
- 会话式认证机制
- 数据本地加密存储

### 🎨 用户体验
- 响应式设计，适配所有设备
- 流畅的动画过渡效果
- 直观的操作界面
- 拖拽式文件上传
- 视频播放控制（倍速、全屏）

## 📊 数据管理

### 支持的文件格式
| 类型 | 格式 | 限制 | 建议尺寸 |
|------|------|------|----------|
| 图片 | JPG, JPEG, PNG | 无数量限制 | 400×300px |
| 视频 | MP4, MOV | ≤200MB/个 | 16:9比例 |
| LOGO | JPG, JPEG, PNG | 单个文件 | 75×75px |
| 背景 | JPG, JPEG, PNG | 单个文件 | 1920×1080px |

### 数据备份
由于数据存储在浏览器中，建议定期：
1. 导出重要数据
2. 备份整个项目文件夹
3. 在不同浏览器中同步使用

## ❓ 常见问题

### Q: 数据会丢失吗？
A: 数据存储在浏览器LocalStorage中，清除浏览器数据会丢失，建议定期备份。

### Q: 支持多少内容？
A: 理论上无限制，但受浏览器存储空间限制（通常5-10MB）。

### Q: 可以多人管理吗？
A: 可以，但需要共享管理密码，建议定期更改。

### Q: 如何迁移数据？
A: 复制整个项目文件夹即可，数据存储在浏览器中，需要重新上传。
